# Gandha Siri 🌳

## Overview

Gandha Siri is an Android application designed to assist sandalwood farmers in managing and protecting their plantations. The app provides digital tree management, GPS-based location tracking, security alerts, multilingual support, and farming guidance through a simple and user-friendly interface.

---

## Problem Statement

Sandalwood farmers often face challenges in plantation monitoring, security management, record keeping, and accessing cultivation guidance. Manual methods are time-consuming and increase the risk of data loss and theft. Gandha Siri provides a digital solution to simplify plantation management and improve security.

---

## Objectives

* Digitally manage sandalwood plantation records.
* Track tree locations using GPS.
* Improve plantation security through SOS alerts.
* Provide farming guidance and legal information.
* Support multiple languages for better accessibility.

---

## Features

### 🌱 Tree Management

* Add and store tree details.
* Record age, girth, and location.
* Upload tree images.

### 📍 GPS Tracking

* Fetch and save tree coordinates.
* View plantation location information.

### 🚨 Security System

* SOS emergency alert feature.
* Security monitoring dashboard.
* Notification-based alerts.

### 🤖 Advisor Module

* Provides sandalwood cultivation guidance.
* Answers common farming-related questions.

### 🌐 Multilingual Support

* English
* Kannada

### 📜 Legal Guide

* Information regarding sandalwood cultivation regulations.
* Quick access to official resources.

### 👤 Profile Management

* Store farmer and farm information.
* Persistent user preferences.
